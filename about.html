About Techy

A student-built tech shop concept focused on clean shopping flows

Techy is a demo storefront designed to make browsing gadgets simple, fast, and visually clear. The project focuses on straightforward product discovery, lightweight cart behavior, and a modern interface without backend complexity.

What Techy is built around

Clear browsing

Products, categories, and pricing are easy to scan so shoppers can move through the catalog without friction.

Simple cart flow

The cart keeps state locally and lets users adjust quantities quickly without page reload complexity.

Modern presentation

The UI uses bold product cards, focused spacing, and a consistent visual language across each page.
